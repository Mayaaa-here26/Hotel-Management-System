Hotel Management System

A console-based hotel management system in C++ for managing rooms and bookings. Supports admin and guest operations.

Features

Admin: Login, view rooms, make bookings, checkout, save/load data

Guest: View rooms, make bookings

Handles room availability, bookings, checkout, and data persistence

Installation

Clone the repository:

git clone https://github.com/your-username/hotel-management-system.git


Compile the code:

g++ main.cpp -o HotelManagement


Run the program:

./HotelManagement

Files

admin.txt – Stores admin usernames/passwords

bookings.txt – Stores booking data
